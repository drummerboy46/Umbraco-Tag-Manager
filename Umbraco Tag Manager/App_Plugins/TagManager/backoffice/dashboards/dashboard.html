Tag ALL the things!

Tag Manager features:

  • Create Groups
  • Create Tags
  • Delete Tags - single and multiple
  • Rename Tags
  • Merge Tags
  • View Content and Media associations

Tag List Property Editor features:

  • Content editors, pick from pre-defined tags in Tag Manager (managed by admins).
  • Tags are maintained in the internal and external Examine Indexes (in a simple CSV format).
  • Relationships are maintained for Content and Media and can be viewed in Tag Manager.
  • (Coming soon) - A Pre-Value switch to allow Content Editors to add Tags to the Tag List.
